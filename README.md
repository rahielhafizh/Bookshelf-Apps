This is the final project of the course “Learn to Build Front-End Web for Beginners”. This project uses JavaScript, HTML, and CSS to create an interactive and easy-to-use platform for managing a book collection. The application features DOM manipulation for dynamic content updates, allowing users to add, edit, and delete books in categories such as “Unfinished Reading” and “Completed Reading.” The Web Storage API is used to maintain data persistence, storing all book information locally in the user’s browser without the need for server-side storage.
